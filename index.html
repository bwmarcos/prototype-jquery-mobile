FAQ

Caso o banco de dados firebird estiver corrompido.

COMPACTAR:

gbak -B -V -Z -user SYSDBA -password masterkey

RESTORE:

gbak -R -V -Z -user SYSDBA -password masterkey

GFIX:

gfix -mend -full -ignore -user SYSDBA -password masterkey

STRING MALFORMED:

gbak -R -V -Z -FIX_FSS_M win1252 -user SYSDBA -password masterkey


VERSÃO DO BANCO:

Gstat -h diretório_do_banco

Firebird 1.0 - ODS 10

Firebird 1.5 - ODS 10.1

Firebird 2.0 - ODS 11.0

Firebird 2.1 - ODS 11.1

Firebird 2.5 - ODS 11.2

Conversão de Banco de Dados Firebird 1.5 para Firebird 2.5

Ainda na versão 1.5 efetuar o seguinte comando de Backup:

gbak -B -V -Z -user SYSDBA -password masterkey

Desinstalar o firebird 1.5 e instalar a versão 2.5

Executar o comando de restauração e conversão rodando em firebird 2.5:

gbak -R -V -Z -FIX_FSS_METADATA WIN1252 -FIX_FSS_DATA WIN1252 -user SYSDBA -password
